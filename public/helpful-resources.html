Some useful websites for learning web development:

Codecademy: Has interactive lessons for many web technologies.

Khan Academy: Has video lessons and interactive exercises for many web and programming topics.

W3Schools: This is a good reference for web development. It explains all the HTML tags and attributes, all the CSS selectors and properties, all the JavaScript functions and more.

DevDocs: This site combines all the documentation for different programming languages and libraries in one place. You can search for any JavaScript function, HTML tag, or anything else and it'll instantly show you the documentation. Just be sure to enable docs for the languages you want.

Other useful links:

CatBnB Project: Source code for CatBnB, the most awesome place for our feline friends to rent cardboard boxes.

Glitch: Make web apps, or view examples from the community.
